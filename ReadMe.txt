## 📌 Overview

This is the official website for *Hotel Transylvania Sigiriya, featuring a homepage with an embedded video, a **Facilities* page, a *Contact Us* page, and other essential sections. The website is designed to be fully responsive and provides users with an interactive experience.

## 🏗 Features

- *Responsive Header & Navigation Bar* ✅
- *Full-screen Video Background on Landing Page* 🎥
- *Facilities Page* - Showcases hotel amenities 🏨
- *Contact Form* - Collects user inquiries 📩
- *Preloader & Popup on Load* ⏳
- *Image Gallery & Services Section* 🖼

## 🔧 Technologies Used

- *Frontend*: HTML, CSS, JavaScript
- *Local Storage* (For storing user data temporarily)


### Step : Open in Browser

- If using *Live Server* (VS Code Extension), right-click on index.html and select *"Open with Live Server"*.
- Otherwise, open index.html manually in a browser.

### Step 3: Hosting (Optional)

To deploy the website:

1. Upload files to *cPanel* or any web hosting service.
2. If using *PHP*, make sure your server supports it.
3. If using a database, update *database credentials* in contact.php.
